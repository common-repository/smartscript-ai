An AI-powered WordPress plugin that generates content directly within your post editor using OpenAI's GPT-3 technology.

== Description ==

**SmartScript AI** is a powerful WordPress plugin that integrates OpenAI's GPT-3 technology directly into your WordPress editor. Generate high-quality, unique content effortlessly, saving time and enhancing productivity. Perfect for bloggers, marketers, and content creators looking to streamline their content creation process.

**Features:**

* **Seamless Integration:** Generate AI-powered content directly within the WordPress editor.
* **Multi-Language Support:** Create content in multiple languages with an easy-to-use language selector.
* **Customisable Prompts:** Input custom prompts to tailor the generated content to your needs.
* **Supports Gutenberg and Classic Editor:** Compatible with both the Gutenberg block editor and the Classic editor.
* **Efficient Workflow:** Insert generated content directly into your post, enhancing your writing efficiency.

**Use Cases:**

* **Blog Posts:** Generate ideas, outlines, or entire articles.
* **Social Media Content:** Create engaging social media posts.
* **Product Descriptions:** Write compelling product descriptions for your e-commerce site.
* **Marketing Copy:** Craft persuasive marketing content for emails and landing pages.

**Note:** An OpenAI API key is required to use this plugin. Sign up at [OpenAI](https://openai.com/) to obtain an API key.

== Installation ==

1. **Upload the Plugin:**
   * Upload the `smartscript-ai` folder to the `/wp-content/plugins/` directory.
   * Alternatively, install the plugin through the WordPress plugins screen by searching for **SmartScript AI**.

2. **Activate the Plugin:**
   * Go to the **Plugins** page in your WordPress admin dashboard.
   * Activate **SmartScript AI**.

3. **Configure API Settings:**
   * Navigate to **Settings > SmartScript AI**.
   * Enter your OpenAI API key in the **AI API Key** field.
   * Optionally, specify supported languages in the **Supported Languages** field.
   * Click **Save Changes**.

4. **Use the Plugin:**
   * Create a new post or edit an existing one.
   * In the post editor, locate the **SmartScript AI** meta box.
   * Select your desired language, enter a prompt, and click **Generate Content**.
   * The generated content will be inserted directly into your post editor.

== Frequently Asked Questions ==

= Do I need an OpenAI API key to use this plugin? =

Yes, an OpenAI API key is required to generate AI content. You can obtain one by signing up at [OpenAI's website](https://openai.com/).

= Is there a cost associated with using the OpenAI API? =

OpenAI's API is a paid service. Please refer to [OpenAI's pricing page](https://openai.com/pricing/) for detailed information on costs.

= Does this plugin support multiple languages? =

Yes, the plugin supports content generation in multiple languages. You can select your desired language from the dropdown menu in the meta box. Ensure that the language is supported by OpenAI's GPT-3 model.

= How do I add or remove supported languages? =

Go to **Settings > SmartScript AI** in your WordPress admin dashboard. In the **Supported Languages** field, enter comma-separated language codes (e.g., `en,es,fr`). Save your changes to update the language options.

= Is the generated content unique? =

The plugin uses OpenAI's GPT-3 model to generate content based on your prompts. While the content is generated uniquely each time, it is important to review and edit the content to ensure it meets your quality standards and is free from unintentional duplication.

= How do I insert the generated content into my post? =

After clicking **Generate Content**, the AI-generated content is automatically inserted into your post editor at the cursor's current position.

= Can I use this plugin with the Gutenberg Block Editor? =

Yes, SmartScript AI is compatible with both the Gutenberg Block Editor and the Classic Editor.

= How do I uninstall the plugin? =

To uninstall, deactivate the plugin from the **Plugins** page and then delete it. Your settings and data related to the plugin will be removed.
